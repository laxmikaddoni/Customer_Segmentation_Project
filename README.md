Customer Purchase Behavior Analysis

## Overview
This project analyzes customer purchase data to understand buying behavior, segment customers, and 
provide business-driven marketing recommendations using Python, SQL, Exel, and Power BI.

## Tools Used
Python (Pandas, Scikit-learn, Matplotlib)
SQL
Power BI
Excel

## Key Steps
Data Cleaning & EDA
Discount & Sales Analysis
Customer Clustering (K-Means)
Dashboard Creation

## Customer Segments
High Spenders
Seasonal Buyers
Budget Customers

## Insights
Small discounts boost sales in high-demand categories
Controlled offers improve loyalty
Discounts help convert low-sales but high-potential categories

## Recommendations
Loyalty rewards for high spenders
Seasonal discounts for seasonal buyers
Promotions for budget customers
